## ✨ Features

* **⚡ Fast & Lightweight:** Instant startup and updates thanks to the Vite build tool.
* **📱 Responsive UI:** Perfectly scaled for mobile, tablet, and desktop screens using Bootstrap 5.
* **🔢 Math Logic:** Handles basic operations (Addition, Subtraction, Multiplication, Division) and clear functions.
* **🎨 Modern Design:** Minimalist aesthetics with a focus on usability and readability.
